# Sprint 1 - 需求文档

## 项目概述
Antipode: DayDreamNight 是一个基于 Qt 的全球视频社交原型应用。

## 功能需求

### 用户界面
- [ ] 底部导航栏（5 个页面）
- [ ] 首页 - 视频推送
- [ ] 朋友圈 - 时间线
- [ ] 录制页面 - 相机功能
- [ ] 聊天页面 - 消息列表
- [ ] 个人资料页面 - 用户信息

### 核心功能
- [ ] 自动计算对立点
- [ ] 视频播放
- [ ] 视频录制（10秒限制）
- [ ] 用户交互（点赞、评论、分享）

## 非功能需求
- [ ] 国际化支持（英文/中文）
- [ ] 无障碍支持
- [ ] 高对比度模式
